# Cod Puppeteer
documentação do [Puppeteer](https://github.com/puppeteer/puppeteer)

## Intalar as dependencias
```
#node
npm install

#yarn
yarn
```

## Iniciar o robo
`
node index.js
`

### .env
```
EMAIL=
PASS=
```
